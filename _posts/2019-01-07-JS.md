es5의 문제점을 보완하기 위해 es6에서는 let 과 const 문법을 정의했다.

## let 과 const 의 공통점
  - 중복선언이 불가능하다.
  ```js
  let foo = 1;
  let foo = 2; // Uncaught SyntaxError
  ```
